# evapotranspiration

Implements Hargreaves-Samani, Thornthwaite, and Penman-Monteith reference evapotranspiration (ETP) from basic meteorological inputs.

## References

- Hargreaves, G.H. and Samani, Z.A. (1985). *Reference crop evapotranspiration from temperature.* Applied Engineering in Agriculture, 1(2), 96-99.
- Thornthwaite, C.W. (1948). *An approach toward a rational classification of climate.* Geographical Review, 38(1), 55-94.
- Allen, R.G. et al. (1998). *FAO Irrigation and Drainage Paper No. 56.* FAO, Rome.
