# plot_maps — Cartographic Visualization

Cartopy-based utilities for plotting regional maps with shapefiles, color classifications, and legends.

!!! info "Dependency"
    Requires `cartopy`, `geopandas`, and `mapclassify`. Install system libraries (GEOS, PROJ) before installing cartopy.
