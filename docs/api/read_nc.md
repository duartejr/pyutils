# read_nc — NetCDF4 I/O

Provides helpers for reading gridded climate variables from NetCDF4 files, with optional spatial subsetting to a shapefile bounding box.

!!! info "Dependency"
    Requires `netCDF4` and `geopandas`. Install system libraries (HDF5, GEOS) before `pip install netCDF4`.
