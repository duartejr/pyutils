# pyeto.fao — FAO-56 Functions

Complete implementation of FAO-56 (Allen et al., 1998) procedures for estimating reference evapotranspiration from meteorological data.

## Reference

Allen, R.G., Pereira, L.S., Raes, D., & Smith, M. (1998). *Crop evapotranspiration: guidelines for computing crop water requirements.* FAO Irrigation and Drainage Paper No. 56, FAO, Rome.
